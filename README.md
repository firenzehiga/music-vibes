## Overview
Music Vibes adalah aplikasi web untuk eksplorasi musik, dirancang agar pengguna dapat menemukan lagu terbaru, lagu populer, dan mengelola playlist. Data musik diambil dari API Spotify, memberikan info musik terkini dan otentik.

> **Disclaimer:**  
> Website ini hanya menampilkan daftar lagu dan album dari Spotify, **tidak menyediakan fitur untuk memutar lagu secara langsung**.

## Fitur
- **New Releases:** Menampilkan album dan lagu terbaru dari Spotify.
- **Popular Songs:** Daftar lagu yang sedang trending.
- **Playlist:** Buat, kelola, dan lihat playlist favorit.

## Tech Stack
- **Frontend:** React + Vite
- **Routing:** React Router
- **Styling:** CSS, styled-components
- **API:** Spotify Web API

## Instalasi & Setup

1. **Clone repository**
   ```bash
   git clone https://github.com/firenzehiga/music-vibes.git
   cd music-vibes
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Akses aplikasi**
   Buka browser dan kunjungi `http://localhost:5173`
